# Sales-Data-Analysis-Excel
# Sales Data Analysis using Excel

## 📌 Project Overview
This project involves cleaning, processing, and analyzing a dataset using Microsoft Excel.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Line)

## 📊 Key Steps
- Data Cleaning (removed duplicates, handled missing values)
- Data Processing using Excel functions
- Created Pivot Tables for analysis
- Built a dashboard for visualization

## 📈 Insights
- Identified top-performing categories
- Found sales trends over time
- Analyzed customer behavior patterns
